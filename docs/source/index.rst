Musictheorpy: A music theory library in Python
==============================================

Welcome to Musictheorpy. This project is intended to provide
everything you need to get started developing applications
involving music theory, including notes, intervals, chords,
and scales. Getting started is easy. All you need to do is
run ``pip install musictheorpy`` to get the latest version.
Once you have the the package installed, you can begin coding
immediately. For more information on the library API, proceed
to the API reference.
